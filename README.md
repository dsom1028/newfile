
 
## 통복시장_청년숲
 
평택 통복시장 내 가칭 주단골목이 긴 잠에서 깨어나 ‘청년숲’으로 탈바꿈했다. 청년들이 만들어낸 맛있고 아기자기한 골목의 가게들을 잘모르는 사람들을 위해 소개하려한다 .
